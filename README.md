End-to-End Windows Server IT Support Lab

Objective
To simulate a real-world IT support environment using Windows Server
and multiple client machines.

Tools Used
- Windows Server 2019/2022
- Windows 10 Clients
- VirtualBox

Configurations
- Active Directory Domain Services
- DNS & DHCP
- User and Group Management
- File Server with Permissions
- Password Policies

Troubleshooting Scenarios
- User login issues
- DNS resolution problems
- IP conflicts
- Permission errors

Outcome
Hands-on experience in IT support operations, system administration,
and troubleshooting.
